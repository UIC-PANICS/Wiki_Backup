====== MCP 导师关顾计划 ======

===== 背景介绍 =====


在希腊神话中，Mentor 是位老人，他受托于奥迪西斯 (Odysseus)，成为他儿子的老师和朋友。Mentor 这字的现代解释，是启蒙导师，是亦师亦友、对你忠心、关心的智者。在古希腊史诗《奥德赛》的几个场景中，智慧与技艺的女神阿西娜假以 Mentor 形式给建议予奥迪西斯和他儿子。Mentor 已被广泛地当成 "忠诚"，"可靠" 和 "智慧" 的代名词。

时至今天，大学有 "师友计划"，大企业有 "师友制"，英文都是用 Mentor，其实就是请一位较有经验、年长一点的 "师友"，给新入行的小伙子中肯的意见，带他们多见世面。在学校，老师则以智慧和忠诚关顾，成为学生的 "生活导师"。Mentor 身为过来人，也可以定期聆听年轻人的烦恼，不时 "指点迷津"，让他们不用走这么多冤枉路。当追求知识或面对逆境时，导师的支持使学生受到不同方面的启发而成长。我们今天的梦想可能正是我们导师昔日的经验，信念和价值观。

===== UIC 推行的导师关顾计划 =====

UIC 推行的导师关顾计划 (Mentor Caring Programme, 简称 MCP)，是维系着学生与 UIC 的重要网络，导师作为学校的代表，被期待连接学生和学校，以建立关系。同时，导师亦需分享他们的宝贵经验，培养和推动学生的个人成长，学生在此可获得导师的关怀和指导，加快对 UIC 的国际化教育模式的适应。

新生入校伊始，UIC 全体新生都安排参加 MCP。同学们按其所读的专业分成 MCP 小组，各自有一名来自同一专业的教师担任导师。导师们通过聊天、聚会等多样形式给予新生 关怀和辅导之外，导师亦会分享他们的专业知识和宝贵经验，培养和推动学生的个人成长。同时每个 MCP 小组都配置就读于同一专业的老生担任的 “朋辈导师” (Peer Mentor)，协助导师与学生联系，并且协助举办小组活动，帮助新生适应新的学习环境。

每一位组员（新生）就是学习者，向导师及其他组员学习；是参与者，积极参与及贡献 MCP 组的活动；是这个组的一员，与导师及其他组员建立和谐关系。

=====MCP目标=====
 
  - 促进学校与学生之间的沟通
  - 为学生提供关顾、指导、建议及支持，从而促进学生个人成长及发展
  - 透过导师与组员间的有效沟通，明白学生的困难及需要，从而提供帮助
  - 建立温馨关爱的校园气氛，从而提升学生对学校的归属感
